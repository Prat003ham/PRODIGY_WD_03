# ❌⭕ Tic Tac Toe Web App
An interactive and responsive **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**. This project allows users to play either against another player or against an AI opponent.

# 🚀 Features
* 👥 **Player vs Player Mode**
* 🤖 **Player vs AI Mode (Random AI)**
* 🧠 Win & Draw Detection
* 🔄 Reset Game Functionality
* 📱 Fully Responsive Design
* 🎨 Clean and Interactive UI

# 🛠️ Technologies Used
* **HTML5** – Structure of the game
* **CSS3** – Styling and layout (Grid & Flexbox)
* **JavaScript** – Game logic and interactivity

# 🎮 How to Play
1. Choose a mode:
   * **2 Players** → Play with a friend
   * **Play vs AI** → Play against the computer

2. Players take turns marking cells:
   * Player 1 → ❌ (X)
   * Player 2 / AI → ⭕ (O)

3. The first player to align **3 marks in a row** wins:
   * Horizontally
   * Vertically
   * Diagonally
     
4. If all cells are filled without a winner → **Draw**
